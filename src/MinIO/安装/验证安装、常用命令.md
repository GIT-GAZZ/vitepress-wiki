# 验证安装、常用命令

```shell
# 浏览器访问：http://<HOST>:9001

# 检查minio服务
sudo systemctl status minio

# 启动minio服务
sudo systemctl start minio
# 停止minio服务
sudo systemctl stop minio
# 重启minio服务
sudo systemctl restart minio

# 开启自启minio服务
sudo systemctl enable minio
# 关闭自启minio服务
sudo systemctl disable minio
```
