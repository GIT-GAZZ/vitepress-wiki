# Typora 破解

`计算机\HKEY_CURRENT_USER\SOFTWARE\Typora\IDate` 这个注册表变量是 Typora 试用的过期时间，直接修改会被 Typora 识别出来，需要将 `计算机\HKEY_CURRENT_USER\SOFTWARE\Typora` 这个文件夹的权限关掉
