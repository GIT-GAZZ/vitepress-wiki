# 安装方式
+ CentOS
    - 通过源码编译安装
    - 通过 `rpm` 命令和 `.rpm` 包手动安装（Red Hat Package Manager）：
        * 通过 Yum 导出 RPM 包
        * [手动下载 RPM 包](http://mirrors.aliyun.com/centos/7/os/x86_64/Packages/)
        * 软件包官方网站下载
    - 通过 `yum` 包管理器安装
+ Ubuntu
    - 通过源码编译安装
    - 通过 `dpkg` 命令和 `.deb` 包手动安装
    - 通过 `apt` 包管理器安装
