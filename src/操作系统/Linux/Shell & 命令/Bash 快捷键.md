# 命令编辑
+ `Alt + R`：删除整行字符
+ `Ctrl + U`：删除光标到行首的所有内容。
+ `Ctrl + K`：删除光标到行尾的所有内容。
+ `Ctrl + Y`：粘贴最近删除的内容（配合 `Ctrl + U`、`Ctrl + K`、`Ctrl + W` 使用）。

# 其他操作
+ `Ctrl + C`：终止当前运行的命令或进程。
+ `Ctrl + Z`：将当前进程挂起并放到后台（可以用 `fg` 命令将其恢复到前台）。
+ `Ctrl + L`：清屏（类似于 `clear` 命令）。
+ `Ctrl + R`：启动反向搜索历史命令。
+ `Ctrl + G`：退出反向搜索模式。
+ `Ctrl + P`：显示历史命令中的上一条命令。
+ `Ctrl + N`：显示历史命令中的下一条命令。
+ `Alt + .`、`Esc（松开） + .`：插入上一个命令的最后一个参数。
